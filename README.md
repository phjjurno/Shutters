# 서트레스 SHUTRESS 🔨 — Shut the Stress.

스트레스 유발자(교수님·직장 상사·진상 손님 등 20명)를 골라 명함을 주고받고,
때리고 꼬집고 물을 뿌리며 스트레스를 해소하는 무료 브라우저 게임.

빌드 과정 없는 순수 정적 사이트입니다. 상위 폴더의 wsQf-PDF와는 완전히 독립적입니다.

## 실행 (로컬)

```bash
# Mini Site 루트에서 (wsQf-PDF와 같은 서버 공유)
python3 -m http.server 8931
# → http://127.0.0.1:8931/bbusyeo/ 접속
```

## 구조

```
index.html        # 단일 페이지: 히어로 → 대상 고르기 → 해소존 → 가이드 → 랭킹 → 플레이리스트 → FAQ → 문의
css/style.css     # 디자인 토큰(슬랙 컬러풀·알약 버튼·Inter·라인 아이콘) + 컴포넌트
js/config.js      # 백엔드(Supabase) 키 — 비우면 로컬 전용
js/data.js        # 캐릭터 20명, 도구 5종(랜덤 데미지), 칭호, PULSE ORIGIN 트랙
js/ranking.js     # 전국 통합 랭킹 클라이언트 (집계 숫자만 전송)
js/app.js         # 캐릭터 SVG, 검색/필터, 명함 모달, 해소존 게임 루프, 랭킹, 플레이리스트
privacy.html      # 개인정보처리방침
terms.html        # 이용약관
robots.txt / sitemap.xml / ads.txt / assets/og.svg   # SEO·애드센스
design/           # 두두(DUDU) 캐릭터 원본 스타터 키트 (배포 제외 가능)
```

## 게임 규칙 요약

- **캐릭터를 직접 클릭해야** 데미지. 빈 곳을 누르면 "피했지롱~" 하고 약올림.
- 도구: `1` 때리기(2~4) · `2` 꼬집기(1~3, 잡은 곳이 늘어남) · `3` 간지럽히기(1~2) ·
  `4` 볼 짜부(2~3, 양손 압착) · `5` 붙잡아 오기(데미지 없음, 포획 전용). `Space` 공격, `Enter` 물 바가지(+5).
- **도망**: 가끔 뒤돌아서 뒷모습으로 15초간 도망 — `5`번으로 캐릭터를 직접 클릭해 포획.
  못 잡으면 "(이름)이 도망갔습니다" → 다시 시작.
- **그로기**: 같은 도구로만 12연타 이상 계속 때리면(20초 쿨다운) 3초간 해롱해롱 — 그동안 공격 불가. 도구를 바꾸거나 물바가지를 쓰면 연타가 끊긴다.
- 맞을 때 가만히 있지 않고 살짝씩 자리를 옮겨 피해 다닌다.
- **완전 해소(100%)**: 퉁퉁 부은 캐릭터가 "스트레스 좀 풀렸어?" →
  "아니, 아직!"=한 판 더 / "응, 시원해"=축하 화면.
- 첫 타격에 플레이리스트가 자동 재생(PULSE ORIGIN 트랙 69% 우선).
- 해소존 좌상단 버튼으로 **전체화면** 플레이 지원.
